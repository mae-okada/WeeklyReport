# 週刊レポート生成スクリプト

## 使い方
- レポート生成には **`generate_report.bat`** ファイルをクリックしてください。
- **`output`** フォルダ内に最新のレポートが格納されます。

## 各レポートの定義

### 1. プロジェクト_ステージ変更リスト.txt
- **`data`** フォルダ内に格納されているエクセルファイルのうち、  
  ファイル名が一番大きい **2つのファイル** を比較します。
- 2つのファイルのうち、**`Stage`** の値に変動がみられる行のみをレポート化します。

### 2. プロジェクト_営業部管轄リスト.txt
- **`data`** フォルダ内に格納されているエクセルファイルのうち、  
  ファイル名が一番大きい **1つのファイル** をレポート化します。
- レポート化対象のデータは、**`Owner Fullname`** が営業部員の行のみです。

## 各ステージの条件

- **1-1. Potential (New Opportunity)**  
  新規案件

- **1-2. Potential (Renewal)**  
  更新案件  
  - 「プロジェクト_営業部管轄リスト.txt」内の対象：**来月更新のプロジェクトのみ**

- **2. Proposal**  
  提案

- **3. Quotation**  
  見積り提出

- **4. S/O**  
  注文書受領  
  - 「プロジェクト_営業部管轄リスト.txt」内の対象：**過去7日以内に上がってきたプロジェクトのみ**

- **5. Sales (Invoice)**  
  請求書送付済  
  - 「プロジェクト_営業部管轄リスト.txt」内の対象：**過去7日以内に上がってきたプロジェクトのみ**
